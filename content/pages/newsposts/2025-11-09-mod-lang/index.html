Hytale mods will be made with Java and Json

By: alizardguy | November 9, 2025

A long asked question about Hytale has been "what language will mods be made in?". Simon responded to Twitter user @N3LSONROJ4S asking "So, should I study Luau or Java for mods creation?" with "Java / JSON".

Java / JSON

— Simon (@Simon_Hypixel) November 9, 2025

This confirms that at least the server side modding will be done in Java, as many people have already suspected from previous blog posts.
